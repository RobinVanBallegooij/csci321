Beyond the Horizon

Game written in one week in Python for the PyWeek 20 game programming
challenge, by Team Universe Factory 20 (Christopher Night, Mary Bichner,
Charles McPillan, Leo Stein, Molly Zenobia, and Randy Parcel), Aug 2015.

https://pyweek.org/e/unifac20/

To play:
Open game.exe and follow the instructions.

Keyboard controls:
arrow keys or WASD: move
space, enter, shift, or Z: teleport
backspace: emergency teleport
escape: quit
F5: save game
F11: toggle fullscreen
F12: take screenshot

Open options.txt to set game resolution an other options.
