DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/
  PyOpenGL:   http://pyopengl.sf.net/
  numpy:      http://numpy.scipy.org/
  pysco:      http://psyco.sourceforge.net/ (optional - will increase performance if available on your platform)



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Twist the fields to control the disk. The red fields are twistable, the blue are fixed.

Avoid red walls - these will set you back to your starting position.



CONTROLS:

Left/Right - rotate the red field.
